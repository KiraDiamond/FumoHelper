About

Project Scope

FumoHelper is a utility project for FUMO. This page exists so people can see what the bot is and what it does without exposing the private codebase behind it.

  • Player stats
  • Raid stats
  • Activity tracking
  • Guild contribution leaderboards
  • War counts
  • Guild raid logs

Status

This is an unofficial Discord app for FUMO. It is not an official guild app, and this public repository is only intended to present the project at a high level.

Public repository for project information only. Operational code, deployment details, and internal bot source remain private.
