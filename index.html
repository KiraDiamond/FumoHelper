FumoHelper

FumoHelper is an unofficial Discord app for FUMO. It is built for checking player information, raid activity, guild tracking, and contribution-related stats in one place.

This repository is the public-facing page for the project. The bot source code is not public.

Some features depend on tracked guild-member data, so tracker-based commands are limited to players in the tracked guild.

  • Player stat lookups

  • Guild contribution leaderboards
  • War count tracking
  • Guild raid logs
  • Recent guild action history
Project Status

Main Commands
/player Shows a player card with account overview, classes, and main profile stats.
/rstats Shows a player's raid completions and raid-specific breakdowns.
/activity Shows tracked activity over time, including recent stat gains and playtime. This is a tracker-based command.
/gxp Shows recent or all-time guild contribution leaderboard rankings from tracked guild-member data.
/warcount Shows tracked war counts for a guild member over a selected timeframe.
/graids Shows detected guild raid history for one player from tracked guild-member data.
/graids_recent Lists recently detected guild raids across tracked guild members.
/graids_total Shows a leaderboard of guild members by tracked raid count.
/guild Shows a guild overview with level, members, and related summary information.
/guild_action Shows recent guild joins, leaves, promotions, demotions, and level-ups.
